## What is CurrentPrompt?

CurrentPrompt is Keith's personal knowledge base of structured markdown modules. Each module includes:
- Categorized content with tags
- Multiple versions with changelogs
- Downloadable formats (full markdown, summary, ZIP bundle)
- Integration with Supabase, Webflow, and LLM agents
