# UX Debugger Skill

A comprehensive skill for identifying and resolving user experience issues in web applications and interfaces.

## Overview

The UX Debugger helps you systematically analyze user interfaces to identify pain points, usability issues, and opportunities for improvement. This skill combines best practices from UX research, accessibility standards, and behavioral psychology.

## Key Features

- **Heuristic evaluation** - Apply Nielsen's usability heuristics
- **Accessibility auditing** - Check WCAG 2.1 compliance
- **Interaction analysis** - Evaluate user flows and task completion
- **Visual hierarchy review** - Assess information architecture
- **Error prevention** - Identify potential user mistakes

## How to Use

1. Describe the interface or upload screenshots
2. Specify the target audience and use cases
3. Request specific analysis (accessibility, usability, etc.)
4. Review recommendations and prioritize fixes

## Example Prompts

- "Analyze this checkout flow for usability issues"
- "Check this form for accessibility problems"
- "Evaluate the visual hierarchy of this dashboard"

## Best Practices

- Focus on user goals and tasks
- Consider multiple user personas
- Test with real users when possible
- Prioritize high-impact, low-effort fixes
- Document all findings and recommendations
